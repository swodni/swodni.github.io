然而,笔者在修改这些设置的时候遇到了一个问题.修改后,有的桌面图标的目标位置从C:\Program Files或者C:\Program Files (x86)变为了D:\Program Files (x86)和D:\Program Files .然而,有的桌面图标的目标位置没有发生变化.

目前笔者想不通为什么会出现这种情况,可能是由于部分软件的自我保护机制吧,他们可能会根据环境变量自己改变快捷方式的目标位置.所以我们只能再次为那些软件创建他们的快捷方式.
